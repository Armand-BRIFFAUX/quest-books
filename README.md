# Quest Books ⚔️

Livre dont vous êtes le héros — Application web interactive.

## Stack

- **Front-end** : Vue.js 3
- **Back-end** : Express.js

## Installation

### Server

```bash
cd server
npm install
npm run dev
```

### Client

```bash
cd client
npm install
npm run dev
```
